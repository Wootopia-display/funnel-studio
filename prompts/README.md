# Prompts — Règle centrale

Tous les prompts utilisés dans ce workspace sont stockés ici.

## Règle absolue
Aucun prompt critique ne doit exister en dehors de ce dossier `/prompts/`.
Si un nouveau prompt est créé, il doit être ajouté ici immédiatement.

## Fichiers disponibles
- `workspace-init.md` — prompt de création de ce workspace
- `start.md` — prompt de démarrage utilisateur
- `page-generation.md` — prompt de génération de page
- `lovable.md` — prompt pour Lovable
