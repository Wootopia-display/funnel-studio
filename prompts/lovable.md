# Prompt : Lovable

## Usage
Prompt à envoyer à Lovable pour générer la page visuellement.

## Prompt

```
Build a complete marketing landing page with the following structure and content.

---

## Design system

- Primary color: [COULEUR PRINCIPALE — ex: #1A1A2E]
- Secondary color: [COULEUR SECONDAIRE — ex: #E94560]
- Background color: [FOND — ex: #F9F9F9]
- Font (headings): [TYPOGRAPHIE TITRES — ex: Inter Bold]
- Font (body): [TYPOGRAPHIE CORPS — ex: Inter Regular]
- Style: [STYLE GÉNÉRAL — ex: minimal / dynamic / corporate]

---

## Page sections

### 1. Hero
- Headline: [TITRE PRINCIPAL]
- Subheadline: [SOUS-TITRE]
- CTA button: [TEXTE BOUTON] → [LIEN OU ANCRE]
- Visual: [DESCRIPTION IMAGE OU ILLUSTRATION]

### 2. Problem
- Title: [TITRE DE SECTION]
- Content: [TEXTE — 2 à 4 phrases ou liste de points]

### 3. Solution
- Title: [TITRE DE SECTION]
- Content: [TEXTE — présentation de l'offre]

### 4. Social proof
- Title: [TITRE DE SECTION]
- Testimonials: [LISTE DES TÉMOIGNAGES avec nom + texte]
- Key numbers: [CHIFFRES CLÉS si disponibles]

### 5. Offer
- Title: [TITRE DE SECTION]
- Offer name: [NOM DE L'OFFRE]
- Price: [PRIX]
- Inclusions: [LISTE DE CE QUI EST INCLUS]
- Guarantee: [GARANTIE si applicable]

### 6. CTA
- Title: [TITRE D'APPEL À L'ACTION]
- CTA button: [TEXTE BOUTON] → [LIEN]

---

## Technical requirements

- Fully responsive (mobile-first)
- Semantic HTML5
- Clean, modern UI
- Fast load (no heavy dependencies)
- All CTAs clearly visible above the fold and at the bottom
```

## Format attendu
Le prompt doit inclure : structure de page, copy section par section, directives visuelles.
