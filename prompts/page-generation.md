# Prompt : Page Generation

## Usage
Prompt principal de génération de page marketing complète.

## Prompt

```
Avant de générer la page, vérifie que les éléments suivants sont tous validés :

- [ ] Étape 5 — Stratégie de page choisie (angle + promesse principale)
- [ ] Étape 6 — Structure de page validée (ordre des sections)
- [ ] Étape 7 — Copywriting complété et approuvé section par section
- [ ] Étape 8 — Choix de design validés (couleurs, typographie, style)

Si l'un de ces éléments est manquant ou non validé, arrête-toi et indique ce qui manque.

Si tout est validé, demande-moi une confirmation explicite avant de générer.

Une fois ma confirmation reçue, génère la page complète dans cet ordre :
1. Structure HTML sémantique section par section
2. Contenu copywriting intégré dans chaque section
3. Classes CSS cohérentes avec les choix de design validés

La page doit être exploitable directement, sans correction majeure.
```

## Contexte requis avant utilisation
- Brief projet complété
- Avatar défini
- Positionnement validé
