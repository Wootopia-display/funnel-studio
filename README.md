# Funnel Studio

Workspace de création de pages marketing piloté par Claude Code.

## Rôle
Ce système permet de créer des pages marketing complètes (brief → copy → design → déploiement) via des skills spécialisés et des workflows structurés.

## Démarrage rapide
1. Ouvrir `member-start/start-here.md`
2. Suivre les étapes d'initialisation
3. Utiliser le prompt de démarrage dans `member-start/initial-start-prompt.md`

## Structure
- `workflows/` — workflows d'exécution
- `skills/` — compétences spécialisées
- `templates/` — modèles réutilisables
- `prompts/` — tous les prompts centralisés
- `client-input/` — données client à renseigner
