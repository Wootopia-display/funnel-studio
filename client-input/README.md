# Client Input — Données projet

Ce dossier contient les fichiers propres à chaque projet client.

## Comment l'utiliser
1. Créer un sous-dossier par projet : `client-input/nom-du-projet/`
2. Y copier et remplir le template depuis `templates/project-brief-template.md`
3. Stocker ici tous les fichiers fournis par le client (textes, images, liens)

## Convention de nommage
`client-input/[nom-client]-[date]/`
